**Projet EasyMemberShip**
**Conseils importants**
Chaque membre doit cree sa propre branche et travailler sur cette brache une fois terminer vous allez pusher sur votre branche mais **a noter il faut toujour recuper ce qui se trouve a la brache principale brache main avant de travailler sur votre branche (faite toujours git pull origin main) cela permet d'eviter les conflits**
**Pour avoir le code complet il faut faire git checkout main ensuite git pull origin main apres vous avez tous les mise a jours et vous pouver travailler sur votre branche**
Ne codez jamais directement dans main
Chaque nouvelle taches -> nouvelle branche(git checkout -b nom_branche)
**Apres avoir terminer votre travail sur votre branche il faut faire 
git add.
git commit -m "vous devez preciser ce que vous avez faites pour eviter de se perdre(exemple:ajout de la page de contact)"
git push origin le nom de votre branche**
